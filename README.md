## Quick Start

Clone the project (ssh is reconmmended):

```bash
git clone --recursive git@github.com:SHUSCT/SHU-Computer-Architecture-Experiments.git SHU-CAE

cd SHU-CAE
```

Build required libraries:

```bash
bash ./scripts/build-libs.sh
```
